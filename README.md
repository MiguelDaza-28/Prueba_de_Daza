# Prueba_de_Daza

El programa se ejecuta y lanza 5 opciones distintas:

• Registrar nuevos estudiantes.
• Consultar la lista de estudiantes.
• Buscar un estudiante por algún criterio (ej. ID o nombre).
• Actualizar la información de un estudiante.
• Eliminar estudiantes.

cada una de estas opciones funcionan ya que cada opcion se convirtio en una funcion diferente que se desarrolla teniendo en cuenta el diccionario creado con anterioridad.

Este programa podria ser muy util en ambientes en los que se necesitan estar al tanto de cada estudiante/empleado que tenga una empresa o escuela
